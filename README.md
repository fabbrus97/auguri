#Auguri
Uso: `./fai_auguri [nome festeggiato]`

Stampa una serie di cose a tema "compleanno".

Dipendenze:

`lolcat` (`sudo apt install lolcat`)
`fortune` (`sudo apt install fortune`)
`toilet` (`sudo apt install toilet`)
`cowsay` (`sudo apt install cowsay`)
